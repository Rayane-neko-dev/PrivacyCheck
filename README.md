# PrivacyCheck - Web App

## Overview
PrivacyCheck is a Django web application that simulates a **GDPR compliance evaluation** for a website.  
Users answer a **short questionnaire** about data collection, consent, cookies, and privacy policies.  
The app provides a **compliance rating**, explanations, and suggestions for improvement.  

This project is **educational** and demonstrates the link between **informatics, law, and societal impact**.

---

## Features
- GDPR questionnaire with multiple-choice questions  
- Automatic compliance evaluation (Compliant / Partially Compliant / Non-Compliant)  
- Explanations and recommendations for users  
- Responsive layout using HTML/CSS (Bootstrap or free template)
